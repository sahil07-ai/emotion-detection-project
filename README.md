\# Emotion Detection Project



\## 📌 Description

This project uses IBM Watson NLP to detect emotions from text input.



\## 🚀 Features

\- Detects emotions like joy, sadness, anger

\- Uses Watson Natural Language API



\## 🛠️ Technologies Used

\- Python

\- Requests library

\- IBM Watson NLP



\## ▶️ How to Run

1\. Install requests library

2\. Run emotion\_detection.py

3\. Enter text to analyze emotions
